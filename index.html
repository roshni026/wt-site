Site Technical

01

HTML 5

The HyperText Markup Language or HTML is the standard markup language for documents designed to be displayed in a web browser. We have used html5 to build out website.

02

CSS 3

CSS is the language we use to style an HTML document. CSS describes how HTML elements should be displayed. We have used the latest version of css, CSS3 to design out website.

03

Javascript

JavaScript is a text-based programming language used both on the client-side and server-side that allows you to make web pages interactive. We have used it to provide some interactive features in our website.

04

J-Query

jQuery is a JavaScript library that helps to simplify and standardize interactions between JavaScript code and HTML elements. We have used jQuery to provide hover effects in the site.
